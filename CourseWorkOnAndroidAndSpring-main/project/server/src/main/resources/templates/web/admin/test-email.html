Проверка почты

Статус

Почта настроена. Письма при входе, создании отчёта и смене пароля должны отправляться.

Почта не настроена — письма не отправляются.
Задайте в application.yml (или переменными окружения):
spring.mail.host (например smtp.mail.ru),
spring.mail.username (ваш email),
spring.mail.password (для Mail.ru и Yandex — «пароль приложения», не основной пароль).
Перезапустите приложение после изменения настроек.

Отправить тестовое письмо

Если письмо не пришло, проверьте папку «Спам» и текст ошибки выше (если есть).
