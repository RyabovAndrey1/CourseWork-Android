# Диаграммы курсовой работы (PlantUML)

Диаграммы для пояснительной записки и презентации.

## Файлы

- **use_case_diagram.puml** — диаграмма прецедентов (акторы: Студент, Преподаватель, Администратор факультета, Администратор системы; сценарии просмотра оценок, ведения журнала, управления планами и т.д.).
- **er_diagram.puml** — концептуальная ER-диаграмма сущностей (users, students, teachers, study_groups, subjects, grades, departments, faculties, attendance).

## Соответствие реализации

В приложении используется та же модель предметной области с небольшими уточнениями:

- **Оценки:** в БД связь «преподаватель – группа – предмет» вынесена в таблицу `assigned_courses` (назначенные курсы); в журнале оценок используется `assignment_id` и справочник `grade_types` (лекция, практика, лаб. работа, экзамен и т.п.).
- **Посещаемость:** в таблице `attendance` поля `lesson_date` и `present` (присутствовал/отсутствовал); статусы «по уважительной причине» можно отражать в поле `comment`.
- **Группы:** в таблице `study_groups` при необходимости можно добавить поле `curator_id` (куратор); в текущей схеме его нет.

Для генерации изображений из `.puml` можно использовать [PlantUML](https://plantuml.com/) (онлайн, локально или плагин в IDE).
